## Wipro Preskkill Training Repository

This repository contains the training materials and resources for the Wipro Preskkill Training program. this repo consists of various coding exercises, projects, and during the training process.

### Contents

* **Assignment**: A collection of coding assignments ans ppt files.
* **Projects**: Various project done using C++ and Linux Bash scripting.
* **Resources**: Additional learning materials and references.
* **Classwork**: Code and notes from class sessions.
