## Wipro Preskilling Training Repository

This repository contains the training materials and resources for the Wipro Preskilling Training program. This repo consists of various coding exercises, projects, and during the training process.

### Contents

* **Assignment**: A collection of coding assignments and ppt files.
* **Projects**: Various projects done using C++ and Linux Bash scripting.
* **Resources**: Additional learning materials and references.
* **Classwork**: Code and notes from class sessions.

### Getting Started

To Make use of this repository, clone it to your local machine using the following command:

```bash
git clone https://github.com/Sarabesh-13/wipro-preskilling-training.git
```
