## Wipro Preskilling Training Repository

This repository contains comprehensive training materials and resources from the Wipro Preskilling Training program, covering Computer Architecture, C++ Programming, and Linux System Programming.

##  Repository Structure

This repository is organized into two main folders:
- **Assignments**: Contains all TOC tasks and assignment submissions
- **Classwork**: Contains practice programs and notes from classroom sessions
